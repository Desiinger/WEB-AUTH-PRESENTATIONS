Setting up your site Cache

Local Folder: This is your working directory—usually a folder on your hard drive. Editors (Dreamweaver) refer to this folder as your local site root

This repesents the server

This is where optimised html, css, js, images, videos etc are kept

This is NOT where development files are kept, i.e photoshop files etc

Why Define a local folder?

A common beginner mistake is to start working on web pages in without defining a site. This results in links to assets, such as images and style sheets, pointing to a location on your local computer (cdrive\desktop\wayne\folder) Everything works fine until you come to upload the site to your remote server. Then you discover all your links are broken and need to be fixed.

We reference all assets via relative links (relative to the current page)

linking to an image then becomes "/assets/week1assets/imgname.jpg"

or linking to a style sheets becomes "/css/style.css"

(Makes it much easier to update)

Responsive Web Design

Responsive website design is the most suitable type of web design style and the most popular in today’s time. It provides precise viewing experience of a website to the user, adapting to all screens across devices.

Media queries

Media query is a CSS technique introduced in CSS3. It uses the @media rule to include a block of CSS properties only if a certain condition is true.

i.e If the browser window is smaller than 500px, the background color will change to lightblue:

 
@media only screen and (max-width: 500px) {
    body {
        background-color: lightblue;
    }
}

FTP

The File Transfer Protocol (FTP) is the standard network protocol used for the transfer of computer files between a client and server on a computer network. FTP is built on a client-server model architecture and uses separate control and data connections between the client and the server.

Horizontally Centring inline elements (nav)

HTML


<div class='header'>Header</div>
<div class='nav'>Navigation</div>

CSS


.header {
  width: 100%;
  height: 100px;
}

.nav {
  width: 100%;
  height: 60px;
  background: #ff5252;
  color: #fff;
  position: relative;
  margin-bottom: -60px;
  z-index: 3;
}

.sticky {
  position: fixed;
  top: 0;
}

js


var yourNavigation = $(".nav");
yourHeader = $('.header').height();

$(window).scroll(function() {
  if ($(this).scrollTop() > yourHeader) {
    yourNavigation.addClass("sticky");
  } else {
    yourNavigation.removeClass("sticky");
  }
});
